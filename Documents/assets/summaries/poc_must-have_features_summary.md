## Page 1

POC Must-Have Features
(Based on survey and interviews)
1. One-Tap Dictation → Structured Note
• Seamless dictation that instantly transcribes into text.
• Works reliably within the 5–10 minute golden window after session.
• No need for multiple clicks or manual file handling.
Why: Therapists report typing is the biggest friction; dictation saves time 
and maintains therapeutic presence.
